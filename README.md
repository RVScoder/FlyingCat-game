# Flying Cat Game 🐱

A simple Scratch game where you control a flying cat and try to avoid obstacles.

## Features
- Control a flying cat using keyboard inputs
- Obstacle avoidance gameplay
- Score tracking system
- Collision detection

## How to Play
Link: https://scratch.mit.edu/projects/1111346825
- Click on the green flag to start
- Use the arrow keys (or the w,s keys) to move the cat.
- Avoid hitting obstacles and survive as long as possible.

## Built With
- Scratch 3.0

## Learning Outcomes
- Working with events and broadcasts
- Creating game loops
- Collision detection
- Score tracking
